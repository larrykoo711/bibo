$ Examples

  • bibo "The quick brown fox jumps over the lazy dog."

    Speak text with default voice (Amy)

  • bibo "Welcome to Bibo." -v ryan

    Use a different voice (Ryan - male)

  • bibo -l

    List available voices

  • bibo -d amy

    Download a voice model

  • bibo "Welcome" -o welcome.wav

    Save speech to file

  • cat article.txt | bibo

    Read from stdin
