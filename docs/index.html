$ Examples

  • bibo "Hello, world!"

    Speak text with default voice

  • bibo -l

    List available voices

  • bibo -d amy

    Download a voice model

  • bibo "Welcome" -v amy -o welcome.wav

    Save speech to file

  • cat article.txt | bibo

    Read from stdin

  • bibo -f "Quick announcement" -v danny

    Fast mode (1.2x speed)
